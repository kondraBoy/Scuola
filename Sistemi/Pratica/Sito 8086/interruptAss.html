Interrupt DOS - INT 21h

Torna alla macro area

Un interrupt è un segnale che interrompe il flusso normale di esecuzione di un programma per richiamare una funzione specifica o un'operazione di sistema. È uno dei principali meccanismi utilizzati dai processori per gestire eventi esterni o interni che richiedono attenzione immediata. Gli interrupt permettono al sistema di rispondere a eventi come input/output, errori hardware, o operazioni temporizzate senza dover attendere che il programma principale li gestisca direttamente.



Funzione 01h - Leggere un Carattere da Input Standard

La funzione 01h dell'interrupt INT 21h legge un carattere da input standard (di solito la tastiera). Il carattere letto viene restituito nel registro AL. Esempio: 

            MOV AH, 01h   ; Funzione di lettura carattere
            INT 21h       ; Chiamata di interrupt


Funzione 02h - Scrivere un Carattere su Output Standard

La funzione 02h dell'interrupt INT 21h scrive un carattere su output standard (di solito lo schermo). Il carattere da stampare viene specificato nel registro DL. Esempio: 

            MOV AH, 02h   ; Funzione di scrittura carattere
            MOV DL, 'A'   ; Carattere da scrivere
            INT 21h       ; Chiamata di interrupt


Funzione 09h - Stampa di una Stringa

La funzione 09h di INT 21h permette di stampare una stringa su output standard. La stringa deve terminare con il simbolo '$'. Esempio: 

            MOV AH, 09h         ; Funzione di stampa stringa
            MOV DX, OFFSET msg  ; Indirizzo della stringa
            INT 21h             ; Chiamata di interrupt
        msg db 'Ciao, mondo$', 0


Funzione 4Ch - Terminare il Programma

La funzione 4Ch di INT 21h termina l'esecuzione di un programma e restituisce il controllo al sistema operativo. È possibile restituire un codice di uscita specificato nel registro AL. Esempio: 

            MOV AH, 4Ch   ; Funzione di terminazione programma
            MOV AL, 00h   ; Codice di uscita 0
            INT 21h       ; Chiamata di interrupt


Esempio Completo - Lettura e Stampa di un Carattere

Questo esempio combina le funzioni di lettura e scrittura caratteri per leggere un carattere da tastiera e stamparlo su schermo. Esempio: 

            MOV AH, 01h   ; Funzione di lettura carattere
            INT 21h       ; Legge il carattere e lo mette in AL
            MOV DL, AL    ; Copia il carattere letto in DL
            MOV AH, 02h   ; Funzione di scrittura carattere
            INT 21h       ; Stampa il carattere su schermo
